## Примеры

**Чат и мультимодальность (doubao-general)**

```
Пользователь ❯ Проанализируй скриншот отчёта — какое изменение выручки Q3 год к году?

Doubao       ❯ Согласно скриншоту, выручка Q3 составила 48,7 млрд юаней,
               рост на 27,5% по сравнению с 38,2 млрд годом ранее.
               Основные драйверы: реклама (+35%) и облачные услуги (+22%).
               Валовая маржа улучшилась с 58% до 62%.
```

```
Пользователь ❯ Выдели ключевые пункты PDF-договора в виде таблицы

Doubao       ❯ | Пункт | Содержание | Риск |
               |-------|-----------|------|
               | Ответственность | 0,05% пени в день | Средний |
               | ИС | Заказчик владеет результатами | Низкий |
               | NDA | Бессрочное взаимное обязательство | Низкий |
               | Эксклюзивность | Запрет на конкурентов 3 года | ⚠️ Высокий |
               Пункт эксклюзивности ограничивает будущий бизнес —
               рекомендуем сократить до 1 года или добавить компенсацию.
```

**Генерация изображений (doubao-generate-image)**

```
Пользователь ❯ Создай постер: аллея гинкго осенью, девушка в бежевом пальто
               со спины, кинематографический свет, вертикальный формат

Doubao       ❯ [Генерация вертикального изображения 9:16]
               Создано в 4K: https://ark-cn-beijing.volces.com/...
               ⚠️ Срок действия URL — 24 часа, скачайте вовремя.
```

**Генерация видео (doubao-generate-video)**

```
Пользователь ❯ Создай 5-секундное видео: золотистый ретривер бежит по лугу,
               замедленная съёмка, полуденное солнце

Doubao       ❯ Задача создана: cgt-s6xcyoSh4BRx
               Статус: очередь → выполнение → готово (47с)
               Скачано: output.mp4 (1080p, 5с)
```

## Установка

```bash
npx skills add Trainerchan/doubao-skill
```

После установки активируйте прямо в диалоге:

```
> Проанализируй это изображение через Doubao
> Создай рекламный постер
> Сделай краткое содержание этого PDF через Doubao
```

### Ручная установка

```bash
git clone https://github.com/Trainerchan/doubao-skill.git
cd doubao-skill
cp .env.example .env
# Отредактируйте .env, укажите ARK_API_KEY
pip install volcengine-python-sdk python-dotenv
```

## Под-навыки

| Навык | Возможности | Модели |
|-------|------------|--------|
| 🔥 **doubao-general** | Чат, распознавание, анализ документов/видео/аудио, веб-поиск, вызов функций | doubao-seed-2.0-lite/pro/mini |
| 🔥 **doubao-generate-image** | Текст→изображение, изображение→изображение, серии, слияние | doubao-seedream-5.0/4.5/4.0 |
| **doubao-generate-video** | Текст→видео, изображение→видео, мультимодальные ссылки, синхронный звук | doubao-seedance-2.0/1.5/1.0 |

> Для генерации видео требуется баланс ≥ 200 юаней или ресурсный пакет.

## Переменные окружения

| Переменная | Обязательно | По умолчанию | Описание |
|-----------|:----------:|-------------|----------|
| `ARK_API_KEY` | ✅ | — | API-ключ Volcengine Ark ([получить](https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey)) |
| `DOUBAO_CHAT_MODEL` | ❌ | `doubao-seed-2-0-lite-260428` | Переопределить модель чата |
| `DOUBAO_IMAGE_MODEL` | ❌ | `doubao-seedream-5-0-260128` | Переопределить модель изображений |
| `DOUBAO_VIDEO_MODEL` | ❌ | `doubao-seedance-2-0-260128` | Переопределить модель видео |

## Как это работает

**1. Маршрутизация** — Родительский навык сопоставляет намерение пользователя с нужным под-навыком.

**2. Самодостаточные под-навыки** — Каждый содержит всё необходимое: проверки, таблицы параметров, примеры кода (cURL + Python), решения ошибок.

**3. Обработка ошибок** — При ошибке API агент автоматически запрашивает актуальную документацию через инструменты поиска или веб-доступа и повторяет попытку.

## Структура репозитория

```
doubao-skill/
├── SKILL.md                      # Родительский навык: конфигурация, маршрутизация
├── REFERENCE.md                  # Справочник: установка, интеграция агентов, повторные попытки
├── .env.example                  # Шаблон переменных окружения
├── doubao-general/
│   ├── SKILL.md                  # Чат и мультимодальность (основное)
│   └── REFERENCE.md              # Расширенные сценарии, таблица параметров, ошибки
├── doubao-generate-image/
│   ├── SKILL.md                  # Генерация изображений Seedream (основное)
│   └── REFERENCE.md              # Потоковая передача, таблица параметров, таблица размеров
├── doubao-generate-video/
│   ├── SKILL.md                  # Генерация видео Seedance (основное)
│   ├── REFERENCE.md              # Расширенные сценарии, таблица параметров, таблица разрешений
│   └── scripts/
│       └── poll_video.py         # Переиспользуемый скрипт опроса и скачивания
├── docs/
│   └── agents/
└── CLAUDE.md                     # Руководство по разработке
```

Основные примеры и быстрый старт — в SKILL.md (≤ 100 строк), полные таблицы параметров и справочники ошибок — в REFERENCE.md. Агенты загружают только необходимое.
